Message Parking::confirmConversation ( Message *  aMessage)

◆ managingConversation()

Message Parking::managingConversation ( Message *  aMessage) const

1e phase de la négociation côté parking.

Durant cette phase, le parking et la voiture

négocient pour tenter de se mettre d'accord sur un prix.

Le parking lit le message reçu (passé en paramètre)

et en crée un nouveau (comme il s'agit forcément d'une réponse, les données du nouveau

message sont initialisées en fonction des données contenues dans le message reçu).

Types de messages générables :

  • OFFER
  • COUNTER_OFFER
  • NO_MORE_SPOTS
  • LAST_OFFER
  • ACCEPT
  • REJECT

Types de messages reçus pris en compte :

  • CALL
  • COUNTER_OFFER
  • LAST_OFFER
  • ACCEPT
  • REJECT
    Parameters
    aMessagemessage reçu
    Returns
    un message
