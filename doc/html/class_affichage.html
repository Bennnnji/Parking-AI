Affiche du texte selon les différents paramètre.

2 choix possible : 1 - Afficher une simple ligne de texte ou 2 - Afficher du texte + une valeur statique ou dynamique (qui change au fil du temps)

Pour passer de du choix 1 a 2 il suffit de mettre juste un String vide ( "" ) sur le parametre Msg ex : AfficherTexte(—, "Son texte", — , —, etc. ) -> Affiche une simple ligne de texte. ex : AfficherTexte(—, "", "Texte", valeur, —, etc.) -> Affiche Texte + valeur

Parameters
fontPolice du texte
Msg1 - Texte simple
MsgWithValeur2 - Texte qui accompagne la valeur
valeurValeur que l'on veut afficher
xPosition en x du texte
yPosition en y du texte
rParametre R du RGBA -> color rouge
gParametre G du RGBA -> color vert
bParametre B du RGBA -> color bleu
aParametre A du RGBA -> transparance
