◆ managingConversation()

Message Voiture::managingConversation ( Message *  aMessage) const

La voiture lit le message reçu (donné en paramètre) s'il y en a un.

(s'il n'y en a pas, le pointeur est nul),

et en crée un nouveau (s'il s'agit d'une réponse, les données du nouveau

message sont initialisées en fonction des données contenues dans le message reçu).

Types de message générable :

  • CALL
  • COUNTER_OFFER
  • LAST_OFFER
  • ACCEPT
  • REJECT

Types de message reçu pris en compte :

  • OFFER
  • COUNTER_OFFER
  • LAST_OFFER
  • ACCEPT
  • REJECT
