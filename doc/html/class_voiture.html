◆ managingConversation()

Message Voiture::managingConversation ( Message *  aMessage) const

1e phase de la négociation côté voiture.

Durant cette phase, la voiture et le parking

négocient pour tenter de se mettre d'accord sur un prix.

La voiture lit le message reçu (passé en paramètre) s'il y en a un

(s'il n'y en a pas, le pointeur est nul), et en crée un nouveau

(les données du nouveau message sont initialisées en fonction des

données contenues dans le message reçu, s'il y en a un).

Types de messages générables :

  • CALL
  • COUNTER_OFFER
  • LAST_OFFER
  • ACCEPT
  • REJECT

Types de messages reçus pris en compte :

  • OFFER
  • COUNTER_OFFER
  • NO_MORE_SPOTS
  • LAST_OFFER
  • ACCEPT
  • REJECT
    Parameters
    aMessage: message reçu
    Returns
    message généré
